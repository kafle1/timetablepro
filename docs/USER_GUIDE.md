## Troubleshooting

### Common Issues
1. Login Problems
   - Check credentials
   - Reset password if needed
   - Contact admin for help

2. Schedule Conflicts
   - Check for double bookings
   - Verify room availability
   - Report to admin

For additional support, use the in-app help system or contact your administrator.
